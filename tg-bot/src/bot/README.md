# Папка bot

Эта папка содержит основную логику Telegram-бота для мониторинга средств индивидуальной мобильности (СИМ).

## Файлы

### `index.ts`

Основной файл для создания и настройки экземпляра Telegram-бота. Выполняет следующие задачи:

- Загружает переменные окружения
- Создает экземпляр бота с использованием `node-telegram-bot-api`
- Настраивает обработку ошибок polling
- Инициализирует все компоненты бота (сервисы, обработчики, middleware, команды)
- Запускает фоновый нотификатор активации курьеров (ActivationNotifier)

### `init.ts`

Файл инициализации зависимостей бота. Создает и настраивает:

- Сервисы (CourierService)
- Обработчики (RegistrationHandler)
- Middleware
- Команды бота

### `state-manager.ts`

Централизованный менеджер состояний пользователей. Управляет:

- Состояниями пользователей (регистрация, выбор склада, взятие/сдача СИМ)
- Временными данными пользователей
- Предоставляет методы для установки, получения и сброса состояний

### `callback-router.ts`

Единый роутер `callback_query` для inline-кнопок:

- Поддерживает регистрацию callback-обработчиков из разных команд
- Централизованно закрывает `callback_query` через `answerCallbackQuery`
- Упрощает поддержку и убирает множественные глобальные `bot.on('callback_query')`

### `admin/admin-mode.ts`

Служебный модуль переключения админского режима:

- Включает/выключает админский режим пользователя
- Блокирует курьерские команды в неавторизованном админском режиме
- Определяет список разрешенных команд до реализации полноценного входа админа

### `keyboards/courier-main-keyboard.ts`

Общий helper для определения и отправки главной reply-клавиатуры курьера по текущему состоянию:

- Определяет, что показать: клавиатуру выбора склада, основную клавиатуру действий или скрытие клавиатуры
- Используется в обработчиках команд для единообразного восстановления UI после отмены/завершения сценариев

### `keyboards/index.ts`

Barrel-файл клавиатур. Экспортирует декомпозированные модули:

- `keyboard.constants.ts` (тексты кнопок и callback-data)
- `registration-flow.keyboard.ts` (клавиатуры регистрации)
- `courier-actions.keyboard.ts` (главные действия курьера)
- `warehouse-selection.keyboard.ts` (выбор склада)
- `sim-selection.keyboard.ts` (выбор СИМ)
- `return-sim.keyboard.ts` (диалог сдачи СИМ)

## Архитектура

Папка `bot` является сердцем приложения и координирует работу всех компонентов:

- **Инициализация**: `index.ts` → `init.ts`
- **Состояния**: `state-manager.ts`
- **Inline callback routing**: `callback-router.ts`
- **Админ-режим**: `admin/admin-mode.ts` + `commands/admin.ts`
- **Команды**: папка `commands/`
- **Обработчики**: папка `handlers/`
- **Middleware**: папка `middlewares/`
