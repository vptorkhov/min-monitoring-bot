# Папка keyboards

## Описание

Содержит функции для создания встроенных reply-клавиатур Telegram бота.

## Файлы

### `registration.keyboard.ts`

Клавиатуры для процесса регистрации:

- **`getRegistrationStartKeyboard()`** - Клавиатура для начала регистрации
  - Содержит кнопку: `/start ✔️ Старт`
  - Используется при запросе имени курьера
  
- **`getCancelKeyboard()`** - Клавиатура для отмены операции
  - Содержит кнопку: `/cancel ❌ Отмена`
  - Используется при вводе имени и телефона

## Использование

```typescript
import { getRegistrationStartKeyboard, getCancelKeyboard } from '../keyboards/registration.keyboard';

// Отправка сообщения с клавиатурой
await bot.sendMessage(chatId, 'Введите имя:', {
    reply_markup: getCancelKeyboard()
});
```

## Особенности

- **Reply keyboard** - встроенная клавиатура, которая появляется вместо стандартной клавиатуры устройства
- **one_time_keyboard** - клавиатура исчезает после нажатия на кнопку
- **resize_keyboard** - клавиатура адаптируется к размеру экрана
- Текст кнопки содержит команду (начинается с `/`), поэтому при нажатии бот обрабатывает это как команду

## Архитектура

Клавиатуры используются в:
- `registration.handler.ts` - обработчик регистрации
