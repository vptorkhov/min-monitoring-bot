# Папка commands/warehouse

Папка содержит команды управления складами в admin-flow.

## Назначение

- Команды выбора/очистки склада для авторизованного администратора.
- Superadmin-команды создания и редактирования складов.
- Вспомогательные функции для резолва выбранного склада и вывода доступных команд.

## Файлы

- `warehouse-commands.ts` — доменный оркестратор склада.
- `warehouse-context-commands.ts` — команды контекста склада для админа (`/admin_set_warehouse`, `/admin_clear_warehouse`).
- `warehouse-superadmin-commands.ts` — команды superadmin для создания и редактирования складов.
- `warehouse-command-helpers.ts` — общие helper-функции для сценариев склада.

## Общий поток

1. Проверка прав (admin или superadmin).
2. Переход в состояние выбора склада или редактирования.
3. Обработка введенного номера/параметров в message-handlers admin-flow.
4. Возврат в корректное авторизованное состояние и вывод актуальных admin-команд.

## Правила изменения

1. Superadmin-операции держать изолированно в `warehouse-superadmin-commands.ts`.
2. Любое изменение состояний сценария синхронизировать с admin message-handlers и `/cancel`.
3. Общие проверки и получение выбранного склада выносить в `warehouse-command-helpers.ts`.
