# Папка commands/sim

Папка содержит команды управления SIM в admin-flow.

## Назначение

- Показ списка SIM на выбранном складе.
- Выбор конкретной SIM.
- Действия с выбранной SIM: статусы, история, удаление.
- Добавление новой SIM.

## Файлы

- `sim-commands.ts` — доменный оркестратор, объединяет list/interactions.
- `sim-list-commands.ts` — список SIM и стартовые сценарии выбора.
- `sim-interaction-commands.ts` — команды действий по выбранной SIM.
- `sim-context-commands.ts` — context-команды для сценариев списка/выбора.
- `sim-base-commands.ts` — базовые SIM-команды admin-flow.

## Общий поток

1. Проверка авторизации admin и выбранного склада (где требуется).
2. Формирование списка доступных устройств.
3. Выбор устройства по номеру/контексту.
4. Выполнение команды действия (изменение статуса, история, удаление).

## Правила изменения

1. Новые команды по SIM добавлять через `sim-commands.ts` как единую точку сборки.
2. Команды, завязанные на выбранную SIM, должны использовать единый контекст выбора.
3. При изменении шагов сценария обновлять message-handlers и cancel-flow.
