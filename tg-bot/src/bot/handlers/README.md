# Папка handlers

Эта папка содержит обработчики диалоговых процессов бота, которые требуют многошагового взаимодействия с пользователем.

## Файлы

### `registration.handler.ts`

Класс `RegistrationHandler` управляет процессом регистрации новых курьеров. Реализует:

- **Двухэтапную регистрацию**: имя → телефон
- **Валидацию данных**: проверка формата телефона
- **Управление состояниями**: использование StateManager
- **Обработку сообщений**: фильтрация команд и обработка ввода

## Архитектура обработчиков

Обработчики:

- Наследуют общий интерфейс для обработки сообщений
- Используют StateManager для управления состояниями
- Интегрируются с сервисами для бизнес-логики
- Предоставляют методы для начала и обработки диалогов
