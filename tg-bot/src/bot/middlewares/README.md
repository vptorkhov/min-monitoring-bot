# Папка middlewares

Эта папка содержит middleware для управления состояниями пользователя и контроля доступа к командам.

## Файлы

### `index.ts`

Центральный файл настройки всех middleware. Выполняет:

- Импорт всех middleware
- Регистрацию middleware в боте
- Координацию порядка выполнения

### `registration-state.middleware.ts`

Middleware для контроля процесса регистрации. Реализует:

- **Блокировку команд** во время регистрации
- **Перенаправление сообщений** в RegistrationHandler
- **Проверку состояний** пользователя

## Архитектура middleware

Middleware в этом проекте:

- Перехватывают входящие сообщения
- Проверяют состояния пользователей
- Контролируют доступ к командам
- Перенаправляют обработку в соответствующие обработчики

## Типы middleware

1. **Состояние-зависимые**: блокируют команды при активных процессах
2. **Валидационные**: проверяют условия выполнения команд
3. **Перенаправляющие**: направляют сообщения в специальные обработчики</content>
