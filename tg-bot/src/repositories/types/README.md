# Папка repositories/types

Папка `repositories/types` содержит общие типы слоя доступа к данным.

## Файлы

- `warehouse.type.ts` — тип/интерфейс сущности склада.
- `session.type.ts` — типы записей сессий и связанных выборок для истории/активных сессий.

## Когда добавлять тип сюда

- Тип используется более чем в одном репозитории.
- Тип описывает контракт между репозиторием и сервисом.

Если тип локальный и используется только в одном файле, допускается держать его рядом с реализацией репозитория.
