# Папка repositories

Папка `repositories` содержит слой доступа к БД (PostgreSQL): SQL-запросы, маппинг строк и типизированные результаты.

## Файлы

- `courier.repository.ts` — операции по курьерам (поиск, создание, активация, привязка к складу, уведомления).
- `warehouse.repository.ts` — операции по складам (чтение, создание, обновление, проверки связанных сессий).
- `mobility-device.repository.ts` — операции по СИМ/устройствам (выборка, поиск, обновление статуса).
- `session.repository.ts` — операции по сессиям (создание, закрытие, активные сессии, история).
- `admin.repository.ts` — операции по администраторам (поиск, регистрация, авторизация, обновление пароля/флагов).
- `types/` — типы, используемые репозиториями.

## Правила слоя

- Только параметризованные SQL-запросы.
- Без бизнес-логики: репозитории возвращают данные, а сценарные решения остаются в `services`/`commands`.
- Типы входа/выхода держите в `types/`, если они переиспользуются между несколькими файлами.
