# Папка config

Эта папка содержит конфигурационные файлы приложения, отвечающие за настройку внешних зависимостей.

## Файлы

### `database.ts`

Модуль конфигурации подключения к PostgreSQL базе данных. Предоставляет:

- **Инициализацию пула соединений**: создание и проверка подключения
- **Повторные попытки подключения**: автоматическое восстановление при сбоях
- **Управление пулом**: синглтон для пула соединений
- **Переменные окружения**: чтение настроек из .env файла

## Архитектура конфигурации

Конфигурационные модули:

- Используют переменные окружения для гибкости
- Предоставляют значения по умолчанию
- Валидируют обязательные параметры
- Реализуют паттерн синглтон для ресурсов

## Переменные окружения

Для базы данных используются:

- `DB_HOST` — хост сервера БД
- `DB_PORT` — порт сервера БД
- `DB_USER` — имя пользователя БД
- `DB_PASSWORD` — пароль пользователя БД
