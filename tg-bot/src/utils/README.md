# Папка utils

Папка `utils` содержит переиспользуемые функции без прямой бизнес-оркестрации.

## Файлы

- `telegram.utils.ts` — извлечение данных Telegram-пользователя, нормализация команд из кнопок.
- `moscow-time.utils.ts` — утилиты работы с датой/временем в московской таймзоне.
- `admin-format.utils.ts` — форматирование текстовых представлений для admin-flow.
- `admin-selection-format.utils.ts` — форматирование списков выбора (склады, СИМ, курьеры и т.д.).
- `admin-state.utils.ts` — вспомогательные проверки/переходы admin-state.
- `admin-message-state.utils.ts` — хелперы для message-router и state-based шагов администратора.
- `admin-status.utils.ts` — функции преобразования и отображения статусов в админских сценариях.
- `cancel-admin-state.utils.ts` — утилиты определения состояния для ветвления `/cancel` в admin-flow.

## Рекомендации

- Утилиты должны быть детерминированными и по возможности чистыми.
- Не добавляйте сюда вызовы репозиториев/сервисов.
- Новые доменные форматтеры именуйте с префиксом области (`admin-*`, `courier-*`) для явной навигации.
