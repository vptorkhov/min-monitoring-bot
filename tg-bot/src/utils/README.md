# Папка utils

Эта папка содержит вспомогательные функции и утилиты, используемые в разных частях приложения.

## Файлы

### `telegram.utils.ts`

Утилиты для работы с Telegram-данными. Предоставляет:

- `extractUserDataFromMessage()`: извлечение данных пользователя из сообщения
- `formatErrorMessage()`: форматирование сообщений об ошибках
- `formatSuccessMessage()`: форматирование успешных сообщений
- `convertKeyboardButtonToCommand()`: преобразование текста reply-кнопок в команды

Поддерживаемые преобразования кнопок:

- `✔️ Старт` → `/start`
- `❌ Отмена` → `/cancel`
- `🏠 Выбрать склад` → `/set_warehouse`
- `🏠Выбрать склад` → `/set_warehouse` (обратная совместимость)
- `🚲 Взять СИМ` → `/take_sim`
- `❌🏠 Отвязаться от склада` → `/clear_warehouse`
- `🚲❌ Сдать СИМ` → `/return_sim`

## Принципы

Утилиты:

- Не зависят от бизнес-логики
- Переиспользуются в разных модулях
- Предоставляют чистые функции без побочных эффектов
- Имеют понятные имена и документацию
