# Папка utils

Эта папка содержит вспомогательные функции и утилиты, используемые в различных частях приложения.

## Файлы

### `telegram.utils.ts`

Утилиты для работы с данными Telegram. Предоставляет:

- **extractUserDataFromMessage()**: извлечение данных пользователя из сообщения
- **formatErrorMessage()**: форматирование сообщений об ошибках
- **formatSuccessMessage()**: форматирование успешных сообщений

## Архитектура утилит

Утилиты организованы по доменам:

- **Telegram**: функции для работы с Telegram API
- **Форматирование**: функции для форматирования сообщений
- **Извлечение данных**: парсинг данных из сообщений

## Принципы

Утилиты:

- Не зависят от бизнес-логики
- Могут быть переиспользованы в разных модулях
- Предоставляют чистые функции без побочных эффектов
