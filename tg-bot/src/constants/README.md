# Папка constants

Эта папка содержит константы и перечисления, используемые в приложении для обеспечения типобезопасности и удобства сопровождения.

## Файлы

### `commands.constant.ts`

Определяет все доступные команды бота:

- **BOT_COMMANDS**: массив всех команд с типизацией
- **isCommand()**: функция проверки, является ли текст командой
- **extractCommand()**: функция извлечения команды из текста

### `states.constant.ts`

Определяет состояния пользователей для управления диалогами:

- **RegistrationState**: состояния процесса регистрации
- **WarehouseState**: состояния выбора склада
- **DeviceSessionState**: состояния взятия/сдачи СИМ

## Архитектура констант

Константы организованы по доменам:

- Команды бота
- Состояния пользователей
- Типы данных

Все константы:

- Используют `as const` для типизации
- Экспортируют типы для TypeScript
