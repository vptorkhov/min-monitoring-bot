# Папка constants

Папка `constants` хранит константы, перечисления и текстовые шаблоны, используемые в разных слоях приложения.

## Файлы

- `commands.constant.ts` — декларация поддерживаемых команд, хелперы проверки и извлечения команд из текста.
- `states.constant.ts` — набор состояний для courier-flow и admin-flow.
- `messages.constant.ts` — централизованные текстовые сообщения/шаблоны, используемые в обработчиках.

## Принципы

- Константы группируются по домену (команды, состояния, сообщения).
- Значения экспортируются как неизменяемые структуры (`as const` там, где это необходимо).
- При добавлении новых команд или шагов диалогов обновляйте `commands.constant.ts` и `states.constant.ts` синхронно.
