# Папка validators

Эта папка содержит функции валидации пользовательских данных.

## Файлы

### `phone.validator.ts`

Валидатор номеров телефонов. Реализует:

- **cleanPhoneNumber()**: очистка номера от форматирования
- **containsOnlyDigits()**: проверка на цифры
- **validatePhoneNumber()**: основная валидация
- **formatPhoneNumber()**: форматирование для отображения

## Архитектура валидаторов

Валидаторы:

- Принимают сырые данные от пользователя
- Возвращают результат валидации с ошибками
- Поддерживают международные форматы
- Предоставляют функции форматирования

## Правила валидации телефона

- Только цифры после очистки
- Длина от 10 до 15 цифр
- Поддержка международного формата
